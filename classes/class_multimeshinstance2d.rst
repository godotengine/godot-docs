Description
-----------

``MultiMeshInstance2D`` is a specialized node to instance a :ref:`MultiMesh<class_MultiMesh>` resource in 2D.

Usage is the same as :ref:`MultiMeshInstance3D<class_MultiMeshInstance3D>`.
