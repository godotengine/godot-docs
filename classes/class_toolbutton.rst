Description
-----------

This is a helper class to generate a flat :ref:`Button<class_Button>` (see :ref:`Button.flat<class_Button_property_flat>`), creating a ``ToolButton`` is equivalent to:

::

    var btn = Button.new()
    btn.flat = true
