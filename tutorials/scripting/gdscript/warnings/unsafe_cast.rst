UNSAFE_CAST
===============

The warning message is:

.. code-block:: none

    Casting "Variant" to "%s" is unsafe.

The default warning level for this warning is **Ignore**.
To modify it, set :ref:`Project Settings > Debug > GDScript > Warnings > Unsafe Cast<class_ProjectSettings_property_debug/gdscript/warnings/unsafe_cast>`.
